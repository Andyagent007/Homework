  1. Explain the following git commands:

    answer

    1. git init is for initialize a new repository of github .
    2. git status is to see every changes that you have made and you haven’t add or committed.
    3. git add is to add a change that you did in the github repository (is more like a change of some file or other thing inside the github repository).
    4. git commit is to save the changes that you did in the repository.
    5. git log is to see every different commit that you have made in the repository.
    6. git push is to push the changes that you made in your terminal into your github account.
    7. git pull is to pull the changes that you have made in your github account by the browser ,to the terminal.
